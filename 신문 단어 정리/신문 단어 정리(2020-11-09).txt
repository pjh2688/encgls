1. 동형암호[homomorphic encryption]
 -> 암호화(encryption)된 데이터를 복호화(decryption) 없이도 연산할 수 있는 암호.
     암호화된 상태에서 연산한 결과값을 복호화하면 평문(plain text) 상태의 데이터를 연산한
     결과와 동일한 값을 도출. 

2.  동형(homomorphic) :  대수학(algebra) 용어인 준동형(homomorphism)에서 유래한 단어로 연산의 구조가 유지되는 특성을 의미.

3. 법인[Corporation] : 법률에 의하여 권리능력이 인정되는 단체 또는 재산'을 의미.

4. New biz - 새로운 사업(신사업)

5. MEC[Mobile Edge Computing]
 -> 무선 기지국에 분산 클라우드 컴퓨팅 기술을 적용하여 다양한 서비스와 캐싱 콘텐츠를 이용자 단말에 가까이 전개함으로써 
     모바일 코어망의 혼잡을 완화하고, 새로운 로컬 서비스를 창출하는 기술이다.

6. RAN[Radio Access Network] : 무선 접속 네트워크.

7. LAN[Local Area Network]  : 로컬 영역 네트워크.

8. 네트워크 엣지[Network Edge]
 -> 인터넷에 연결된 컴퓨터나 다른 장비들은 end system이라고 불리는데 그 이유는 이들이 인터넷의 edge에 위치하기 때문이다.
     end system들은 그들이 웹브라우저나 웹 서버 프로그램 같은 애플리케이션 프로그램을 host(실행)하기 때문에 end system들을 다른 말로 host(호스트)라고도 부른다.
     host는 client(클라이언트)와 server(서버)로 나뉘는데, 클라이언트에는 데스크탑이나 모바일 pc, 스마트폰 등이 있고 서버는 웹 페이지나 비디오 스트리밍 등을 저장하고
     뿌려주는 기계(ex) data center)를 말한다. 최근 우리가 받는 검색 결과나 웹 페이지 등을 뿌려주는 곳의 서버는 보통 아주 큰 data center에 위치한다. 
     예를 들어 구글은 약 십만 개 이상의 서버를 갖는 30~50개의 데이터 센터를 갖고 있다.
